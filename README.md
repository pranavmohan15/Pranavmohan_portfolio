# Pranav Mohan – Portfolio Website

This is my personal portfolio website built to showcase my projects, skills, and background in a simple and focused way.

## 🚀 Live Website
https://pranavmohan-portfolio-pppb.vercel.app/

## 📌 About the Project

The goal of this portfolio is simple:

- Present who I am clearly
- Showcase selected projects
- Provide an easy way to connect
- Keep the design clean and minimal

This is not overloaded with animations or unnecessary elements. The focus is on clarity and usability.

## ⚙️ Installation & Setup

If you want to run this project locally:

### 1. Clone the repository

```bash
git clone https://github.com/pranavmohan15/Pranavmohan_portfolio.git
```

### 2. Navigate into the project folder

```bash
cd Pranavmohan_portfolio
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm run dev
```

The project will run locally on:
```
http://localhost:5173/
```
